    ABC Stories

    AR Handwriting iOS app designed to teach children the alphabet through games and interactive stories

    Roles

    Programmer

    Technical Stack

    Overview

    An iPad application designed to help children learn the alphabet through interactive stories. Featuring a variety of activities, each with unique interactions and animations.

    Showcase

    Original Application

    The previous version of this application was designed in collaboration between the GIMM Works team and teachers and occupational therapists, with the goal of creating an interactive curriculum based off what is currently being taught in the classroom.

    The application has letter imitation, letter matching, free draw, and the various story-based activities. In the story-based activities, the user can help the characters by tracing the corresponding letter, further progressing the story and reinforcing the learning of the alphabet.

    Updates

    We were tasked with updating the application to use local storage instead of a server, as well as adding new UI and menus, and barriers to the activity tracing for the teachers and therapists to see where the students are continually struggling.

    Challenges

    The biggest challenge that I ran into was with the tracing activities. The original application was designed with a specific amount of "points" per line that the user would have to trace through. This worked well for the original application, but made it difficult to build upon. I had to create various solutions to this problem, trying to find the best way to determine when and where the user went off path.

    Ultimately, I was able to create a version of the barrier system that utilized the same points, but recorded when the user moved off the alpha of the underlayed image. This came with many of it's own problems, but I was able to find working solutions to those as well.

    After the initial barrier system was created, I coded the program to place red dots on the screen where the user went off path. When each letter and symbol is completed, a screenshot is taken and saved to the device's local storage. This allows the teacher or therapist to review the student's progress at any point in time.
